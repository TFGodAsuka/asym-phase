# Asym \ phase Web Project

This folder is the root directory of the project website.

## Main Pages
- index.html        : Home page
- overview.html     : Project overview
- posters.html      : Poster system page
- models.html       : Model showcase page
- script.html       : Script page
- future.html       : Future development page

## Style / Interaction
- css/style.css     : Main stylesheet
- js/main.js        : Main JavaScript file

## Asset Folders
- assets/posters/   : Final posters and poster iterations
- assets/models/    : Character renders and scene rebuild images
- assets/process/   : Blender screenshots, design process, experiments
- assets/script/    : Script-related supporting visuals
- assets/video/     : Demo videos and turntable videos
- assets/covers/    : Homepage cover and section banners

## Docs
- docs/             : Source documents, notes, scripts, reports
